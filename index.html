BFB v2

Basic Front Boilerplate (ou BFB, para os íntimos) é a maneira mais fácil de se iniciar um projeto front-end para testes e estudos.

Um boilerplate que leva performance a sério, levemente opinativo, baseado no Parcel, que permite usar recursos de Web moderna hoje mesmo.

Estrutura Robusta

Arquitetura ITCSS para organização dos estilos de maneira eficiente e profissional através do uso de Sass (com a nova sintaxe @use/@forward).

Base de códigos sadia com o uso de code formatters e linters, adotando convenções internacionais para padronização de commits e versionamento.

Gostinho de Futuro

Comece a usar hoje mesmo CSS Logical Properties (padding-block, margin-inline etc.), CSS Color Module Level 4 e Level 5 (lab(), lch(), color-mix() e mais) e Media Queries Level 4 e Level 5 com Custom Media Queries, com a novíssima sintaxe de range ( @custom-media --modern (30em <= width <= 50em)).

JavaScript à vontade

Transpilação de JSX, TypeScript, CoffeeScript e Flow out of the box.

Suporte a ES modules, CommonJS, Dynamic import e Workers -- web workers, service workers e worklets (incluindo paint worklets de CSS Houdini ).

Extensível as F*ck

Centenas de possibilidades para extensão através de PostCSS e seus plugins, com Babel para transformações experimentais ou personalizadas.

Habilitado de fábrica conversor automático de px para rem e com opcional ativável para CSS Cascade Layers (@layer).
